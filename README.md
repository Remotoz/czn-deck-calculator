## How to Use

1. Select your tier (1-15)
2. Enter your deck composition:
   - Neutral cards (20 value each)
   - Forbidden cards (20 value each)
   - Monster cards (80 value each)
3. Add epiphanies (10 value each)
4. Adjust for card operations (copies, removals, conversions)
5. Click **Calculate** to see if you're within the save cap
